Descrição do Projeto

PokeExplorer é uma aplicação web interativa desenvolvida com Next.js 14 que permite aos usuários explorar e descobrir informações detalhadas sobre Pokémon.

A aplicação consome dados da PokéAPI, uma API pública e gratuita que fornece informações completas sobre todos os Pokémon. Através do PokeExplorer, é possível:

  • Visualizar uma listagem dos primeiros 20 Pokémon com imagem e nome
  • Navegar para a página de detalhes de cada Pokémon
  • Consultar informações como tipo(s), peso, altura e habilidades
  • Uma interface intuitiva e responsiva em todos os dispositivos
